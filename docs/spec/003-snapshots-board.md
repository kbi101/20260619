# 003 Snapshots Board — Technical Specification

## Overview
Traders require an independent dashboard window to display image snapshots generated by other applications during the day. In addition, the dashboard must allow traders to upload new snapshots (via clipboard copy/paste, file drag-and-drop, or manual upload), browse historical date folders, view consolidated weekly timelines, and duplicate historical snaps to the active day.

---

## Requirements

### 1. Snapshot Folder Resolution
- Snapshots are stored locally on the host machine at `/Users/kepingbi/Data/QuantEdge/[YYYYMMDD]`.
- The application dynamically resolves today's date in local time.
- If today's directory does not exist, the UI shows an empty state offering direct upload/paste triggers and watches the parent folder for today's folder creation.

### 2. Category & Timestamp Parsing
- Snapshots are files named with the pattern: `[category]_[timestamp].png`.
  - Example: `volatility_201207.png` maps to category `"volatility"` and timestamp `"20:12:07"`.
- Filename parsing:
  - Extract the category name (capitalized in UI).
  - Extract the timestamp.
  - Sort snapshots chronologically.

### 3. Snapshot Ingestion (New Uploads)
- **Clipboard Pastes (Cmd + V):** The board listens globally for paste actions. Intercepts native clipboard screenshot buffers and image files copied from Finder (natively reading and decoding paths).
- **Drag & Drop:** Dragging an image over the board displays a glassmorphic target overlay. Dropping parses the file.
- **Manual Upload:** Add button in Categories panel header or empty state.
- **Smart Categorization Modal:** Any ingest triggers a modal showing a thumbnail preview, autocomplete dropdown of categories, and filename. All files are normalized to `.png`.

### 4. Historical Browsing & Duplicate to Today
- **Date Selector Dropdown:** Glassmorphic dropdown in Categories sidebar containing all historical date folders discovered. Selecting a date updates the board.
- **Consolidated Week View:** Toggle *"Show entire week (Last 7 days)"* combines and sorts snapshots from the last 7 available dates into a single timeline.
- **"Copy to Today" Button:** Viewer canvas displays a button when browsing past dates. Duplicates the image to today's folder via the categorization modal. Automatically redirects the view to Today upon success.

### 5. Interactive Viewport
- Display the selected image centered on a dark canvas.
- Provide a toolbar with:
  - **Copy to Today** (for historical snaps)
  - **Zoom controls**: `+` and `-` scaling from 50% to 300%.
  - **Aspect fit modes**: `Fit`, `Fill`, and `Original` (with scrollbars).

### 6. Desktop Integration & Sizing
- Launcher button in Workspace title bar and application menu (`View -> Show Snapshots Board`). Shortcut `CmdOrCtrl+3`.
- Default window dimensions optimized for 2560x1440 macOS workspaces:
  - **Workspace Window:** 2560x1440
  - **Intel Dashboard:** 1600x1000
  - **Snapshots Board:** 1920x1080

## Architecture & IPC Specification

### 1. IPC Invocation Channels

- `snapshots:list` -> Returns metadata array of snapshots for the given date.
  ```typescript
  interface SnapshotMeta {
    filename: string;
    category: string;
    timestamp: string; // HHMMSS
    mtime: number;
    date?: string; // Date folder string
  }
  ```
- `snapshots:read` -> Reads local binary file, returning a base64 encoded data URL:
  ```typescript
  (filename: string, date?: string) => Promise<string>
  ```
- `snapshots:save` -> Saves base64 encoded image to today's folder:
  ```typescript
  (payload: { category: string; filename: string; base64Data: string }) => Promise<void>
  ```
- `clipboard:read-image` -> Reads native screenshot buffers or Finder copied file references on the clipboard:
  ```typescript
  () => Promise<string | null>
  ```
- `snapshots:get-dates` -> Discovers and returns a list of YYYYMMDD subfolders:
  ```typescript
  () => Promise<string[]>
  ```

### 2. IPC Events
- `snapshots:updated` -> Broadcasts the updated file metadata list to the renderer whenever today's folder contents change.
